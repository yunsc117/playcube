PLAYCUBE

3×3×3 RUBIK'S CUBE — FREE BROWSER GAME

PlayCube is a free, browser-based 3D Rubik's Cube puzzle game. Solve the classic 3×3×3 cube directly in your browser — no download, no sign-up, no plugins required. Built for desktop and mobile alike.

Play Now

Features

3D WebGL Rendering Powered by Three.js with smooth 60 fps animation
Swipe to Move Drag a face to rotate that layer — just like a real cube
2-Finger Orbit Rotate the camera view with two fingers on mobile
Timer & Move Counter Track your solve time and number of moves
Scramble One-tap randomization with 22 random moves
Keyboard Support U D L R F B keys for speed-solving on desktop

How to Play

  • Swipe a face — drag your finger (or mouse) across a face to rotate that layer in the drag direction
  • 2-finger drag — pinch or drag with two fingers to rotate the camera view (mobile)
  • Drag background — click and drag the background area to rotate the camera (desktop)
  • Keyboard U D L R F B — clockwise moves; hold Shift for counter-clockwise
  • Scramble — applies 22 random moves to randomize the cube
  • Reset — returns the cube to solved state and clears the timer

About the Project

PlayCube is a personal project created as a fun, accessible way to play the classic Rubik's Cube puzzle online. The game runs entirely in the browser using WebGL — there is no server-side processing of your gameplay data.

The site is free to use and supported by Google AdSense advertising. No account or registration is ever required.

Technology

  • Three.js — 3D WebGL rendering engine
  • HTML5 / CSS3 / Vanilla JS — no frameworks, fast load
  • Pointer Events API — unified mouse and touch handling
  • Hosted on GitHub Pages at playcube.site
